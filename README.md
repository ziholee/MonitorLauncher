# Monitor Launcher v1.0

Windows 다중 모니터 환경에서 프로그램을 특정 모니터에서 실행할 수 있게 해주는 경량 유틸리티입니다.

## 주요 기능

- **모니터 감지 및 선택**: 연결된 모든 모니터를 자동 감지하고 선택할 수 있습니다.
- **프로그램 실행 위치 제어**: 실행 파일을 선택한 모니터에서 실행할 수 있습니다.
- **프로필(즐겨찾기) 기능**: 프로그램 경로, 모니터, 창 상태를 프로필로 저장하여 원클릭으로 실행할 수 있습니다.
- **창 상태 제어**: 최대화, 일반 크기, 복원 옵션을 지원합니다.

## 주요 파일 구조

~~~
MonitorLauncher/
├── MonitorLauncher.csproj  # 프로젝트 파일
├── app.manifest            # 관리자 권한 요구 설정
├── Program.cs              # 진입점
├── MainForm.cs             # 메인 UI 및 로직
├── Win32Api.cs             # Win32 API 래퍼
├── WindowController.cs     # 창 위치 제어 로직
├── Profile.cs              # 프로필 저장/로드
└── README.md               # 문서
~~~

## 요구사항

- Windows 10 또는 Windows 11
- .NET 8.0 Runtime
- 관리자 권한 (일부 프로그램 제어를 위해 권장)

## 빌드 방법

```bash
dotnet build MonitorLauncher.csproj -c Release
```

## 실행 방법

1. 빌드된 실행 파일을 실행합니다.
2. 관리자 권한으로 실행하는 것을 권장합니다 (일부 프로그램의 창 제어를 위해).
3. 모니터를 선택하고 실행할 프로그램을 지정한 후 "실행" 버튼을 클릭합니다.

## 프로필 저장

1. 프로그램 경로, 모니터, 창 상태를 설정합니다.
2. "프로필 저장" 버튼을 클릭하고 이름을 입력합니다.
3. 저장된 프로필을 더블클릭하면 해당 설정으로 프로그램이 실행됩니다.

## 프로필 저장 위치

프로필은 다음 위치에 저장됩니다:
`%APPDATA%\MonitorLauncher\profiles.json`

## 기술 스택

- C# / .NET 8.0
- Windows Forms
- Win32 API (P/Invoke)

## 주의사항

- UWP(스토어 앱)는 완벽하게 지원되지 않을 수 있습니다.
- 일부 프로그램은 스플래시 스크린으로 인해 창 위치 제어가 지연될 수 있습니다.
- 관리자 권한으로 실행되는 프로그램을 제어하려면 런처도 관리자 권한으로 실행해야 합니다.
